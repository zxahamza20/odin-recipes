# odin-recipes
1st Project of the Foundations of Web Dev course on The Odin Project

Purpose: This project is a collection of recipe web pages created as part of The Odin Project curriculum, demonstrating foundational HTML skills and basic web development concepts. It features a clean, functional recipe website with three distinct dishes from different culinary traditions.

Structure:

Home Page:
index.html - The homepage with navigation to all recipes

Recipe directory:
biryani.html - Detailed recipe for South Asian Biryani
pad-kee-mow.html - Detailed recipe for Thai Drunken Noodles (Pad Kee Mow)
brownies.html - Detailed recipe for Fudgy Brownies dessert

Key features:
  Semantic HTML Structure: 
    Each page uses proper HTML5 semantic elements including headings, paragraphs, lists, and images.

Organized Content Layout:
  Clear recipe title with descriptive image
  Detailed description sections
  Well-categorized ingredient lists with subheadings
  Numbered step-by-step instructions
  Consistent navigation back to homepage

Content Variety: 
  The recipes showcase different cuisines:
    Biryani: A fragrant South Asian rice dish with spiced meat or vegetables
    Pad Kee Mow: A bold Thai stir-fried noodle dish with aromatic herbs
    Brownies: A rich, fudgy chocolate dessert with various customization options

Accessibility Considerations:
  Alt text for all images
  Proper heading hierarchy (h1, h2, h3)
  Clear link descriptions

Visual Elements:
  Recipe-specific images with appropriate dimensions
  Consistent styling through semantic markup
  Logical content grouping

Learning Objectives Demonstrated
  This project showcases practical application of:
    Basic HTML document structure
    Text formatting and semantic markup
    Working with images and alt text
    Creating ordered and unordered lists
    Implementing internal linking between pages
    Organizing content with proper heading hierarchy
    Building multi-page website navigation
    Using git to constantly track and update to github
